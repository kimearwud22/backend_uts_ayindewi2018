What is backend development?

Imagine you are sitting in a car with a rich user interface in front of you. The dashboard, pedals, buttons, touchscreen surround you. Using these visual parts, you can control and drive the vehicle. But these parts alone cannot do the job. They need indispensable parts the parts which are under the hood. The backend. Backend extends the Frontend capabilities that it cannot be done alone.

Backend Development at the GDF


Backend Development at GDF focuses on frontend facing backend solutions the so-called Backend For Frontends or BFFs. A BFF purpose is to provide relevant data to a specific frontend journey application in a clean and well documented way. To make the ownership, the code and the documentation clean, one BFF should only serve one frontend application. Having this clean separation is essential to be able to make independent product design decisions in the customer facing frontend layer. Creating a BFF solution relies on intensive communication between different knowledge experts like product owners, frontend and internal service experts. This close collaboration is essential to fully understand how a frontend application collects and visualizes data provided by the customers.
